设计规范总结

🎨 配色方案

  • 背景：深蓝渐变 (#0f2027 → #203a43 → #2c5364)
  • 玻璃卡片：rgba(255,255,255,0.1) + blur(10px)
  • 边框：rgba(255,255,255,0.15)
  • 文字：白色主文字 + 半透明次要文字
  • 强调色：蓝、绿、紫、橙（状态色）

📐 布局规范

  • 容器宽度：max-width 1600px
  • 间距：16px / 24px（卡片间距）
  • 圆角：16px（大卡片）/ 8px（小元素）
  • 内边距：16px / 20px（卡片内部）
  • 网格：4列指标卡 + 3:1主内容布局

✨ 动效规范

  • 悬停：背景色变化 (0.2s ease)
  • 卡片：上浮 + 发光边框 (0.3s ease-out)
  • 按钮：缩放反馈 (0.1s)
  • 状态：颜色过渡 (0.2s ease)
  • 加载：脉冲 + 渐变流动

🔧 技术实现

  • CSS：Tailwind CSS + 自定义类
  • 玻璃效果：backdrop-filter: blur()
  • 渐变：linear-gradient + 多色停止
  • 动画：transition + @keyframes
  • 响应式：保持移动端可用性
