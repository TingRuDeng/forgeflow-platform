# Run Result Contract v1

- Scope: `run-result-v1`
- Compatibility: additive-only changes are allowed within v1.

Required fields:

- `command`
- `task_id`
- `artifact_root`
- `decision`
- `terminal_state`
- `provider_success_count`
- `provider_failure_count`
- `findings_count`
- `parse_success_count`
- `parse_failure_count`
- `schema_valid_count`
- `dropped_findings_count`
- `created_new_task`
